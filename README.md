# Install

1) git clone
2) npm install
3) install hugo

# Run

1) gulp
2) hugo serve

# New Post

1) hugo new blog/[post name].md


# Deploy

1) gulp build
3) FTP to bradorego_bradorego@ssh.phx.nearlyfreespeech.net
4) Upload the /public/ directory to the /public/ directory on remote
